VARIABLES

it is container for storing data in js we can declare variables in 4 ways --automatically --var --let --const

CASE RULES

CAMEL CASE - FIRST LETTER OF THE FIRST WORD WILL BE SMALL AND REMAINING WILL BE CAPITAL

EXAMPLE-- java Script

PASCAL CASE - EACH AND EVERY WORD FIRST LETTER WILL BE IN CAPITAL

EXAMPLE-- Java Script

SNAKE CASE - EACH AND EVERY WORD IS SEPERATED BY UNDERSCORE( _ )

EXAMPLE-- java_script

VARIABE SYNTAX

var_type var_name = value/data
examples:-
var firstname="sai"
let marks=78
const num=39;

VARIABLES AND RULES

IT ALLOWS ONLY a-z,A-Z,0-9

Variables are not start with numbers

IDENTIFIERS---- UNIQUE NAMES ARE CALLED AS IDENTIFIERS.

ALL JS VARIABLES MUST BE IDENTIFIED WITH UNIQUE NAMES

IDENTIFIERS CAN BE SHORT(X,Y,Z.) NAMES OR DESCRIPTIVE NAMES(PRICE,AGE,VOLUME..)

THE GENERAL RULES FOR CONSTRUCTING NAMES FOR VARIABLES(UNIQUE IDENTIFIERS)
--NAMES CAN CONTAIN LETTERS,DIGITS,UNDER SCORES AND DOLLARS
--NAMES MUST BE BEGIN WITH LETTERS
--NAMES CAN ALSO BEGIN WITH $ OR _
--NAMES ARE CASE SENSITIVE
--RESERVED WORDS CANNOT BE USED AS NAMES
--NAMES CANNOT BE START WITH NUMBERS.
--we cannot use any otherspeecial characters except $ and_
