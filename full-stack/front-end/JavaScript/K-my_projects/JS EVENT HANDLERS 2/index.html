HTML EVENT HANDLER THROUGH ATTRIBUTES WITH JS CODE WE ARE GNG TO ADD EVENTS
MOUSE EVENTS
EVENT DESCRIPTION
ONCHANGE------AN HTML ELEMENT HAS BEEN CHANGED
ONCLICK-------USERS CLICK THE HTML ELEMENTS
ONMOUSEOVER---THE USERS MOVES THE MOUSE OVER THE HTML ELEMENT
ONMOUSEOUT----THE USERS MOVES THE MOUSE AWAY FROM THE HTML ELEMENT
ONMOUSEUP---THE USERS MOVES THE MOUSE OVER THE HTML ELEMENT
ONMOUSEDOWN----THE USERS MOVES THE MOUSE AWAY FROM THE HTML ELEMENT
ONMOUSEMOVE---THE USERS MOVES THE MOUSE OVER THE HTML ELEMENT

KEYBOARD EVENTS
EVENT DESCRIPTION
ON KEYDOWN-----THE USERS PUSH KEYBOARD KEY DOWN
ON KEYUP-----THE USERS PUSH KEYBOARD KEY UP

FORM EVENTS
EVENT HANDLERS DESCRIPTION
ONFOCUS-----------------WHEN USER FOCUSES ON AN ELEMENT
ONSUBMIT----------------WHEN THE USER SUBMITS THE FORM
ONBLUR------------------WHEN THE FOCUS IS AWAY FROM THE ELEMENT
ONCHANGE----------------WHEN THE USERS MODIFIES OR CHANGES THE VALUE OF A FORM ELEMENT

WINDOW OR DOCUMENT EVENTS
EVENT HANDLERS DESCRIPTION
ONLOAD------------------THE BROWSER HAS FINISHED THE LOADING
onunload----------------when visitor leves the current webpage
onresize--------------when visitor resizes the window of the object

EVENT HANDLERS CAN BE USED TO HANDLE AND VERIFY USER INPUT,USER ACTIONS AND BROWSER ACTIONS
--THINGS THAT SHOULD BE DONE EVERY TIME A PAGE LOADS
--THINGS THAT SHOULD BE DONE EVERY TIME A PAGE IS CLOSED
--ACTION SHOULD BE PERFORMED WHEN A USER CLICK ON BUTTON
---CONTENT SHOULD BE VERIFIED WHEN A USER INPUTS THE DATA.....
