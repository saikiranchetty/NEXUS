STRING METHODS

SLICING THE STRINGS

slice means a part. [] operator is called slice operator used to retrieve the part of the strings
-> in strings flows zero base index
-> the index values can be either + or - values
-> positive values means it will start from forward direction from left to right
-> negative values means it will start from backward direction from right to left

slicing syntax

[start:end:step]
-> starting index value is included
-> ending index value is not included

case scenarios

(++) ending value should be greater than starting value
(-+) ending value should be greater than starting value
(--) ending value should be lessthan starting value
(+-) ending value should be less than starting value

+/-===

(+-) ending value should be greater than starting value
(-: ==) ending value should be less than starting value

+/-

(:+==) starting value should be less than ending value
(:-==) starting value should be greater than ending value
