    1. RGB
      1. Red
      2. Green
      3. Blue
        4. The RGB model defines colors based on the combination of Red, Green, and Blue values, each ranging from 0 to 255. We can set these colors using the style attribute directly in HTML elements or through CSS.
    2. HSL
      1. Hue
      2. Saturation
      3. Lightness
        4. Hue: The angle on the color wheel (0 to 360 degrees). 0 is red, 120 is green, and 240 is blue.
        Saturation: The intensity of the color (0% to 100%). 0% is a shade of gray, and 100% is the full color.
        Lightness: The brightness of the color (0% to 100%). 0% is black, 50% is neither dark nor light, and 100% is white.

    3. HEX
      1. Hexadecimal Values(hex)
        2. Hex Color Codes: Hex color codes start with a # followed by six hexadecimal digits. The first two digits represent the red component.
        The middle two digits represent the green component.
        The last two digits represent the blue component.
